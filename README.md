# Conservation Economics - 2026 Quiz App

A website to practise Conservation Economics - 2026.

## Features
- Fully interactive flashcard quiz engine.
- Supports randomization of questions and multiple-choice options dynamically per user.
- Dark mode compatibility for late-night studying.
- Fast, resilient Single Page Application statically powered by React, TypeScript, and Vite.
- Automatically stores settings (like timer options and theme) via IndexedDB so preferences aren't lost on refresh.

## Development

```bash
npm install
npm run dev
```

To build for production:
```bash
npm run build
```
